## Install

Для быстрой установки загрузите последний релиз для Windows со страницы [releases page](https://github.com/AndreyMospanov/Data_Sculptor/releases).
Запустите DataSculptor-Setup.exe и пользуйтесь установленной программой на своём ПК.

Для использования Питона:
Установите необходимые библиотеки:

```powershell
python -m pip install -r requirements.txt
```
Запустите графическую оболочку:

```powershell
python -m core.gui
```
или двойным кликом [data_sculptor_gui.pyw](data_sculptor_gui.pyw).
Если `.pyw` ничего не открывает, запустите [start_data_sculptor_gui.bat](start_data_sculptor_gui.bat) в этой же папке, чтобы посмотреть ошибки в консоли.
Убедитесь, что установлены все дополнительные библиотеки для python.

## Desktop GUI

В приложении несколько вкладок (могут дополняться):

- `Batch rename` предпросмотр и переименование по маске, в т.ч. с использованием дат.
- `XLSX to CSV` массовая конвертация файлов `.xlsx`. Можно выбрать разбелитель для csv Файла.
- `XLSX editor` проверка и редактирование структуры файлов `.xlsx`. Работа с колонками, выбор нужных листов и файлов.
- `CSV editor` аналогичный функционал для `.csv` файлов.

Используйте `Language` меню, чтобы переключить язык.

Во время конвертации GUI показывает текущий файл, прогресс для него, а также кнопку `Cancel` для остановки процесса конвертации пакета файлов в текущей папке.
Также в нижнейй  части GUI для каждой вкладки есть сообщение о статусе выполняемых действий.
